\doxysubsection{Detailed Description}
Calculate matrix multiplication of two square matrices $\overline{\text{AB}} = \overline{\text{A}} \ \overline{\text{B}}$. 

This matrix can be defined on a 3-\/D grid or constant. The storage convention described in \doxylink{structeq__vars_1_1eq__2__type}{eq\+\_\+vars.\+eq\+\_\+2\+\_\+type} is used.
